Образовательная платформа

Платформа JUBEL является автономным веб-приложением для использования в образовательных целях. Главная задача платформы упрощение процесса образования и перенесения всего в цифровую оболочку

Для Высших Учебных заведений

Платформа может перенсти весь подаваемый материал на свой ресурс для облечения процесса преподавания для студентов.

Для школ

Удобная платформа для учеников, родителей и преподавателей. Больше не будет ничего, что можно упустить.
